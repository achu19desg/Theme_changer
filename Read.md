# 🌗 Theme Changer

A simple Theme Changer project created using HTML, Tailwind CSS, and JavaScript.  
This project switches between Light Mode and Dark Mode when the button is clicked.

---

# ✨ Features

- Light Mode and Dark Mode
- Dynamic background color changing
- Button color changing
- Text color changing
- Responsive centered layout
- Beginner-friendly JavaScript project

---

# 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript

---

# 📂 Project Structure

```plaintext
theme-changer/

├── index.html

```

# 🚀 How to Run

1. Download or clone the project
2. Open the project folder in VS Code
3. Run `index.html` using Live Server
